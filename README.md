# Reverse Agent Homework

This repository contains two homework subprojects.

- Currently completed: `angr-react-agent`
- Pending: `binary-react-static-agent`

The completed project demonstrates a ReAct-style orchestration loop that calls
angr tools to inspect a crackme, find its success path, and solve its input.

See [`angr-react-agent/README.md`](angr-react-agent/README.md) for setup and
execution instructions.
